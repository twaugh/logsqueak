- Python is a high-level programming language
  - Known for readability and simplicity
  - Created by Guido van Rossum in 1991
    - Released as open source
- Key features
  - Dynamic typing
  - Automatic memory management
  - Extensive standard library
    - Batteries included philosophy
- Common use cases
  - Web development with [[Django]] and [[Flask]]
  - Data science and [[Machine Learning]]
  - Automation and scripting
  - Scientific computing
- Popular frameworks
  id:: 65a2c1f0-1234-4567-89ab-cdef01234567
  - [[Django]] for web applications
  - [[Flask]] for microservices
  - [[FastAPI]] for modern APIs
  - [[Pytest]] for testing
- Package management
  - pip is the standard package installer
  - Virtual environments isolate dependencies
    - Use venv or virtualenv
- Best practices
  - Follow PEP 8 style guide
  - Write docstrings for documentation
  - Use type hints for clarity
