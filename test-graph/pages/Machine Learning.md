- Machine Learning is a subset of artificial intelligence
  - Systems learn from data without explicit programming
  - Improves performance with experience
- Types of learning
  id:: 65a2c1f0-def0-4567-89ab-cdef0123456a
  - Supervised learning
    - Training with labeled data
    - Examples: classification, regression
  - Unsupervised learning
    - Finding patterns in unlabeled data
    - Examples: clustering, dimensionality reduction
  - Reinforcement learning
    - Learning through trial and error
    - Agent receives rewards or penalties
- Common algorithms
  - Linear regression for prediction
  - Decision trees for classification
  - Neural networks for complex patterns
  - K-means for clustering
- Popular [[Python]] libraries
  - [[Scikit-learn]] for traditional ML
  - [[TensorFlow]] for deep learning
  - [[PyTorch]] for research and production
  - Pandas for data manipulation
- Model evaluation
  - Train/test split
  - Cross-validation
  - Metrics: accuracy, precision, recall, F1
    - Choose metric based on problem type
- Challenges
  - Overfitting and underfitting
  - Data quality and quantity
  - Feature engineering
  - Computational resources
