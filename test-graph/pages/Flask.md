- Flask is a lightweight [[Python]] web framework
  - Micro-framework philosophy
  - Minimal dependencies out of the box
- Key characteristics
  - Simplicity and flexibility
  - Easy to get started
  - Extensible through plugins
    - Flask-SQLAlchemy for database
    - Flask-Login for authentication
    - Flask-WTF for forms
- Routing
  id:: 65a2c1f0-9abc-4567-89ab-cdef01234569
  - Decorator-based route definitions
  - Support for URL parameters
  - HTTP method handling
- Template rendering
  - Uses Jinja2 template engine
  - Template inheritance
  - Macros and filters
- When to choose Flask over [[Django]]
  - Smaller projects
  - Need more control over components
  - Microservices architecture
  - Learning web development
- Common extensions
  - Flask-Migrate for database migrations
  - Flask-CORS for cross-origin requests
  - Flask-Caching for performance
